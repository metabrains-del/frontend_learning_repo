# Frontend Learning Repo

This repository is dedicated to learning and practicing frontend web development, specifically HTML, CSS, and JavaScript.

## Folder Structure
- `html/`: Contains HTML exercises and templates.
- `css/`: Contains styling stylesheets.
- `javascript/`: Contains JavaScript logic and scripts.

## Best Practices for Writing Clean HTML Code

1. **Use Semantic Tags**: Use tags that convey meaning like `<header>`, `<footer>`, `<main>`, `<article>`, and `<section>` instead of just `<div>`. This improves accessibility and SEO.
2. **Proper Indentation**: Consistently indent nested elements to make the code easier to read and maintain.
3. **Include `alt` Attributes**: Always include descriptive `alt` attributes for images to aid screen readers and provide context when images fail to load.
4. **Valid Document Structure**: Ensure your document has `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>` tags correctly nested.
5. **Keep It Lowercase**: Write tags and attributes in lowercase for consistency and compatibility.
6. **Use Comments Sparingly but Effectively**: Add comments to divide large sections or explain complex code, but don't over-comment obvious structures.
7. **Separate Structure and Style**: Keep HTML for structure and use external CSS for styling. Avoid inline styles test.
